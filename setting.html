Dies ist eine Alpha Version, die Editierbaren Einstellungen kommen mit der v.0.8

Anleitung

Bilder Ändern

Header-Bild:
Das Header Bild ersetzt mann wenn man im Ordner img/big-logo.png die datei ersetzt und die neue Datei mit dem selben namen benent.

Icon-Bild:
Die Icon bilder ersetzt mann wenn man im Ordner img/small-logo.png die datei ersetzt und die neue Datei mit dem selben namen benent.

Overview

Die Overview ist das ganz Linke Symbohl im Header. Diese ist kann in der webdesk.html datei bearbeitete werden
in der const data = {} ist der Inhalt der Overview definiert.

Gruppe:
Im String group: " ",

Link:
Unter dem String link: " ", kannst du eine Webseite oder html Datei definieren die aufgerufen werden soll

Button:
Im onClick: " ", String definiertst du die Funktion die ausgefürt wird.
Standartmäsig ist die Funktion "iframeWindow" definiert die den vorher definierten Link im Fenster voll anzeigt
Wenn kein onClick definiert ist wird aus dem Button nur ein Link

                    {
                        group: "Demo Gruppe",
                        panels: [
                        {
                            buttonText: "Button",
                            content: "Das ist ein Button",
                            link: "https://bit-tag.com",
                            onClick: "iframeWindow",
                        },
                        {
                            buttonText: "Link",
                            content: "Das ist ein Link",
                            link: "https://startpage.com",
                        },
                        ],
                    },
Design

Das Design wird in der styles.css Datei definiert.
In der :root scheife kann man Farben, Rundungen, abstände, etc. ändern.

Darkmode:
Den Darkmode ändert man in der zweiten schleife [data-theme="dark"]

Start App

Die App die sich beim starten öffnet, wird zuzeit in der Funktion generateOverview(data) definiert und muss in der webdesk.html manuell geändet werden
