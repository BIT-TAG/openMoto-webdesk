Anleitung

Bilder Ändern

Header-Bild:
Das Header Bild ersetzt man, indem man im Ordner img/big-logo.png die Datei ersetzt und die neue Datei mit dem selben Namen benennt.

Icon-Bild:
Die Icon-Bilder ersetzt man, indem man im Ordner img/small-logo.png die Datei ersetzt und die neue Datei mit dem selben Namen benennt.

Overview

Die Overview ist das ganz linke Symbol im Header. Diese kann in der webdesk.html-Datei bearbeitet werden.

Gruppe:
Im String group: " ",

Link:
Im String link: " ", kannst du eine Webseite oder HTML-Datei definieren, die aufgerufen werden soll.

Button:
Im String onClick: " ", definierst du die Funktion, die ausgeführt wird.

{
    group: "Demo Gruppe",
    panels: [
    {
        buttonText: "Button",
        content: "Das ist ein Button",
        link: "https://bit-tag.com",
        onClick: "iframeWindow",
    },
    {
        buttonText: "Link",
        content: "Das ist ein Link",
        link: "https://startpage.com",
    },
    ],
},
Design

Das Design wird in der styles.css-Datei definiert.

Darkmode:
Den Darkmode ändert man in der zweiten Schleife [data-theme="dark"].

Start App

Die App, die sich beim Starten öffnet, wird in der Funktion generateOverview(data) definiert und muss manuell geändert werden.
